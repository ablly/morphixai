## 1. Tripo3D API 定价体系

### 1.1 两种付费方式

Tripo3D 有两种不同的定价体系：

| 方式 | 平台 | 定价 | 适用场景 |
|------|------|------|----------|
| **Web App 订阅** | studio.tripo3d.ai | $15.9-$111.9/月 | 个人用户在网页上使用 |
| **API 按需付费** | platform.tripo3d.ai | **$1 = 100 积分** | 开发者通过 API 调用 |

### 1.2 API 按需付费定价 (你的项目使用这个)

```
充值比例: $1 USD = 100 Tripo 积分
单价: 1 Tripo 积分 = $0.01 USD
```

**重要**: 你的 Morphix AI 通过 API 调用 Tripo3D，所以成本按 **$0.01/积分** 计算。

## 2. API 功能积分消耗详情

### 2.1 3D 生成基础功能

| 生成类型 | 无纹理¹ | 标准纹理² | 高清纹理³ | 成本范围 |
|----------|---------|-----------|-----------|----------|
| 文字转3D (Text to 3D) | 7 | 10 | 20 | $0.07-$0.20 |
| 图片转3D (Image to 3D) | 7 | 20 | 30 | $0.07-$0.30 |
| 多视角转3D (Multi-view)⁴ | 7 | 20 | 30 | $0.07-$0.30 |

**注释：**
1. 无纹理：设置 `texture=False` 和 `pbr=False`，仅输出基础模型
2. 标准纹理：默认设置，包含烘焙纹理模型和 PBR 模型
3. 高清纹理：设置 `texture_quality=detailed`，更高分辨率
4. 多视角不支持 Turbo-v1.0 模型版本

### 5.3 问题发现与修复

#### ❌ 严重问题：部件分割定价 (已修复)

```
原定价: 5 Morphix积分
Tripo成本: 40 Tripo积分 = $0.40

问题分析:
- 用户购买 Standard 包: $29.99 = 300 Morphix积分
- 原定价下，用户可以做 60 次部件分割 (300÷5)
- 60 次 × 40 Tripo积分 = 2400 Tripo积分 = $24 成本
- 你只收了 $29.99，利润只有 $5.99 (20%)

修复后:
- 新定价: 20 Morphix积分
- 用户可以做 15 次部件分割 (300÷20)
- 15 次 × 40 Tripo积分 = 600 Tripo积分 = $6 成本
- 利润: $23.99 (80%)
```

#### ⚠️ 中等问题：骨骼绑定定价 (已修复)

```
原定价: 10 Morphix积分 = $1.00 (按Standard包)
Tripo成本: 25 Tripo积分 = $0.25
利润: $0.75 (75%) - 可接受但偏低

修复后:
新定价: 15 Morphix积分 = $1.50
利润: $1.25 (83%)
```
