# 变更日志

## 2025-12-06

### 初始化
- 创建项目任务追踪文件
- 初始化 Next.js 15 项目
- 配置项目结构
- 实现 Hero 3D 组件 (粒子变换效果)
- 设置全局黑色背景
- 修复构建错误：降级 Next.js 至 15.0.3 以解决 SWC/Turbopack 兼容性问题
- 升级 Hero 组件：使用 GLSL Shader 实现高性能液态粒子效果
- 视觉优化：大幅提升亮度，增强粒子炸裂范围和液态流动速度
    - **美学升级**：调整为自然光照 (Right-to-Left)，消除"晃眼"感
    - **粒子质感**：使用 Curl Noise 模拟"漂浮"和"液态"效果，而非单纯炸裂
    - **色彩调整**：应用户要求，将粒子颜色调整为**科技感淡蓝色** (Tech Pale Blue)，更具未来感
    - **法线连接**：在重组阶段显示法线线条，增加科技感
    - **序列重构**：实现 5 阶段循环 (实体 -> 点粒子 -> 绝对星云 -> 液态波浪 -> 实体)
    - **Bug修复**：彻底修复"青色团块" (Blob)，通过强制隐藏 Group 实现
    - **节奏调整**：炸裂过程延长至 45% 滚动行程，Lerp 系数降至 0.01，极度缓慢柔和
    - **平滑过渡**：Shader 中引入 `smoothstep` 非线性插值，消除起止突变
    - **移除法线**：应用户要求移除重组阶段的法线线条，保持纯净粒子感
- 交互升级：实现全屏滚动交互 (ScrollControls)
    - 滚动控制模型状态 (实体 -> 漂浮 -> 液态 -> 重组)
    - 滚动控制相机运镜 (Zoom In)
    - **流畅度优化**：使用 Lerp 插值消除状态切换闪烁
    - **背景优化**：添加星空 (Stars) 和微粒 (Sparkles) 增强深邃感
- 国际化 (i18n)：集成 `next-intl`
    - **多语言支持**：完整支持英文 (EN) 和中文 (ZH)
    - **一键切换**：在 FixedUI 中添加语言切换按钮
    - **路由适配**：实现 `/[locale]` 路由结构和中间件重定向
- **交互式 3D 查看器 (Viewer3D)**：
    - **多模式渲染**：支持 Solid (实体), Wireframe (线框), Points (点云), Liquid (液态) 四种模式切换
    - **全能控制面板**：悬浮 UI 支持模式切换、光照强度调节、自动旋转开关
    - **独立演示页**：新增 `/demo` 路由用于独立展示查看器功能
- UI 完善：
    - **FixedUI**：添加顶部导航栏和底部页脚 (Glassmorphism 风格)
    - **内容增强**：扩充 LandingOverlay，包含功能特性、技术参数、价格方案等详细内容
- 新增组件：`LandingOverlay.tsx` (落地页内容), `FixedUI.tsx`
