# Morphix AI - 3D Generation Studio UI Design

## 项目背景
Morphix AI 是一个高端 AI 3D 模型生成平台，面向海外专业用户（设计师、游戏开发者、3D艺术家）。
设计风格：赛博朋克 + 极简科技感 + 暗黑主题

## 设计要求

### 整体风格
- 主色调：深黑背景 (#050505) + 青色高亮 (#22d3ee) + 紫色渐变 (#a855f7)
- 玻璃拟态效果 (glassmorphism)：半透明面板 + 模糊背景 + 微光边框
- 霓虹发光效果：按钮和重要元素带有柔和的发光阴影
- 字体：等宽字体用于数据显示，无衬线字体用于标题
- 动效：流畅的过渡动画，悬停时的微妙缩放和发光

### 页面布局 (Create Page - 3D生成工作室)

#### 左侧控制面板 (400px宽，可折叠)
设计一个垂直排列的控制面板，包含以下模块：

**1. 生成模式选择器 (Generation Mode)**
- 4个模式卡片，2x2网格布局
- 每个卡片包含：图标 + 模式名称 + 基础积分消耗
- 模式列表：
  - 📷 单图转3D (Image to 3D) - 10-25积分
  - ✏️ 文字转3D (Text to 3D) - 10积分
  - 🖼️ 多视角转3D (Multi-view) - 15积分
  - 🎨 涂鸦转3D (Doodle) - 10积分
- 选中状态：青色边框发光 + 背景高亮
- 未选中状态：暗灰边框 + 半透明背景

**2. 输入区域 (Dynamic Input Area)**
根据选择的模式动态显示：

A) 文字输入模式：
- 大型文本输入框，支持多行
- 占位符提示："Describe your 3D model in detail..."
- 字符计数器
- AI 建议标签（可点击快速填充）

B) 单图上传模式：
- 拖拽上传区域，虚线边框
- 支持预览已上传图片
- 图片要求提示（推荐尺寸、格式）
- 删除/替换按钮

C) 多视角上传模式：
- 6个图片槽位，3x2网格
- 每个槽位显示序号和角度提示（正面/侧面/背面等）
- 已上传数量指示器："3/6 images uploaded"
- 最少2张，最多6张的提示

D) 涂鸦模式：
- 简单的画布区域或图片上传
- 提示用户上传简笔画/草图

**3. 质量选择器 (Quality Level) - 仅图片模式显示**
- 3个水平排列的选项卡
- Standard (512px) - 10积分
- High (1024px) - 15积分  
- Ultra (2048px) - 25积分
- 选中状态带有渐变背景

**4. 高级选项面板 (Advanced Options)**
- 可展开/折叠的手风琴设计
- 5个开关选项，每个带有积分标签：
  - 🎨 HD Texture (+5积分) - 高清纹理
  - ✨ PBR Material (+3积分) - PBR材质贴图
  - 🦴 Auto Rigging (+10积分) - 骨骼绑定
  - 📐 Smart Low-poly (+3积分) - 智能减面
  - 🧩 Part Segmentation (+5积分) - 部件分割
- 每个选项带有简短说明tooltip
- 开关使用青色/紫色渐变

**5. 输出格式选择 (Export Format)**
- 水平排列的格式标签
- GLB (Web) | OBJ (Raw) | FBX (Game) | USDZ (AR) | STL (Print)
- 选中状态高亮

**6. 费用计算器 (Cost Calculator)**
- 实时显示总积分消耗
- 分项明细（可展开）：
  - Base: 10 credits
  - + HD Texture: 5 credits
  - + Rigging: 10 credits
  - ─────────────────
  - Total: 25 credits
- 用户当前余额显示
- 余额不足时显示警告 + 快捷充值按钮

**7. 生成按钮**
- 大型渐变按钮 (青色→紫色)
- 霓虹发光效果
- 禁用状态：灰色 + 无发光
- 加载状态：脉冲动画 + "Processing..." 文字
- 图标：闪电⚡或魔法棒🪄

#### 右侧预览区域 (剩余空间)

**1. 3D 视口 (3D Viewport)**
- 占据主要空间
- 深色网格背景
- 等待状态：显示占位符图形 + "Awaiting Input" 文字
- 生成中状态：
  - 中央加载动画（旋转的几何体或粒子效果）
  - 进度条 + 百分比
  - 阶段提示文字（"Analyzing image...", "Generating mesh...", "Applying textures..."）
- 完成状态：
  - 可交互的3D模型预览
  - 鼠标拖拽旋转
  - 滚轮缩放

**2. 视口控制栏 (Viewport Controls)**
- 浮动在视口右上角
- 渲染模式切换：实体/线框/点云
- 光照强度滑块
- 自动旋转开关
- 全屏按钮
- 重置视角按钮

**3. 下载浮动按钮**
- 模型生成完成后显示
- 浮动在视口底部中央
- "Download .GLB" 带下载图标
- 点击后可选择其他格式

#### 底部终端日志 (可选，可折叠)
- 模拟终端风格的日志输出
- 等宽字体，青色文字
- 显示生成过程的实时状态
- 可滚动查看历史日志

### 响应式设计
- 桌面端：左右分栏布局
- 平板端：控制面板可折叠到侧边
- 移动端：垂直堆叠，视口在上，控制在下

### 微交互动效
- 模式卡片悬停：轻微上浮 + 边框发光
- 开关切换：弹性动画
- 按钮点击：涟漪效果
- 进度更新：数字滚动动画
- 模型加载完成：淡入 + 轻微缩放

### 空状态设计
- 未上传图片时：虚线边框 + 上传图标 + 提示文字
- 未生成模型时：3D占位符图形 + "Your model will appear here"
- 积分不足时：警告图标 + 充值引导

### 成功/错误状态
- 成功：绿色提示 + 勾选图标 + 模型预览
- 错误：红色警告框 + 错误信息 + 重试按钮
- 积分不足：黄色警告 + 快捷购买链接

## 设计交付物
1. 完整的 Create 页面设计稿（桌面端）
2. 各状态的设计变体（空状态、加载中、成功、错误）
3. 组件细节设计（按钮、卡片、开关、输入框）
4. 响应式布局示意
